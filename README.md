# Nextjs-tutorial

This repository is about how to build nextjs web application.

## usage

- development

```sh
git clone git@github.com:jayhanjaelee/nextjs-tutorial.git
npm install
npm run dev
```

- production

```sh
git clone git@github.com:jayhanjaelee/nextjs-tutorial.git
npm install
npm run build && npm run start
```
